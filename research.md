# AI-Powered Product Research

> Контекст: Олег (консультант) указал на проблему — "Все 13 продуктов AI-built, но ни один не AI-powered."
> Задача: проработать 2 варианта через формулу WifiCard, выбрать лучший.

---

## Вариант A: Roast My Listing

### WifiCard

| Элемент | Значение |
|---------|----------|
| **Персона** | Настя, 28 лет. Продаёт б/у вещи на Avito/Etsy. Сделала 5 объявлений, 0 откликов за неделю. Не понимает почему не покупают. |
| **Функция** | Вставь ссылку или текст объявления — получи жёсткий но конструктивный разбор: что отпугивает покупателей и как переписать. |
| **Враг** | Сейчас: спрашивает подругу или перечитывает сама. Подруга говорит "нормально". Платные инструменты (PriceLabs $20/мес, AutoRank, Selloquence) — для профи-продавцов, не для Насти. |
| **Эмоция** | "Почему никто не покупает?!" — фрустрация + неуверенность в своём тексте. |
| **Результат** | Конкретный список: 5-7 пунктов что исправить + переписанный вариант объявления. Копипастнул — выложил — продал. |

### Конкуренты

1. **Roast My Web** (roastmyweb.com) — только сайты, не объявления
2. **PriceLabs Listing Optimizer** — Airbnb only, $20+/мес, для хостов
3. **Selloquence** — SEO-оптимизация листингов для Etsy/Amazon, подписка
4. **3Dsellers** — eBay AI tools, enterprise
5. **Bugsmash Roast** — только UX/UI сайтов

**Прямого конкурента "roast объявления на маркетплейсе" — НЕТ.**

### Изюм

- "Roast" формат: не скучный "анализ", а жёсткий разбор с юмором
- Универсальный: Avito, Etsy, eBay, Airbnb, OLX — любой текст
- Бесплатный, без регистрации, single-file HTML
- Результат = готовый текст для копипаста

### Вирусный потенциал: ВЫСОКИЙ

- "Roast" формат доказано виральный (Roast My Web, #AIRoastMe в TikTok)
- Share-момент: "посмотри как меня разнесли!" — скриншот результата
- Двойной вирус: жертва показывает roast + друзья проверяют свои объявления
- Контент для соцсетей: блогеры-ресейлеры могут делать серии "roasting my listings"

### Техническая реализация

- **Вариант 1 (BYOK):** Пользователь вставляет API key Claude/OpenAI. Prompt engineering на клиенте. Один вызов API = один roast. Стоимость ~$0.01-0.03 за roast.
- **Вариант 2 (Gemini API free tier):** Google Gemini API имеет бесплатный tier (15 RPM). Можно хардкодить ключ для демо или использовать OAuth.
- **Вариант 3 (минимальный бэкенд):** Cloudflare Worker (бесплатно, 100K запросов/день) как прокси к API. Один файл worker.js.
- **Рекомендация:** Вариант 1 (BYOK) как основной + Вариант 3 как "try free" режим.

## Вариант B: PriceTag

### WifiCard

| Элемент | Значение |
|---------|----------|
| **Персона** | Дима, 32 года. Разбирает гараж, хочет продать старый велосипед и кучу барахла. Понятия не имеет, сколько просить. |
| **Функция** | Сфотографируй вещь — получи рекомендуемую цену + обоснование на основе рынка. |
| **Враг** | Сейчас: гуглит "сколько стоит б/у велосипед Giant Talon 2019", скроллит 20 объявлений, усредняет в голове. 30 минут на вещь. |
| **Эмоция** | "Не хочу продешевить, но и не хочу чтобы висело месяц" — страх + лень. |
| **Результат** | Число: "Продай за 15,000 руб." + диапазон + 3 аналога из рынка. |

### Конкуренты

1. **PriceSnap** (pricesnap.ai) — бесплатный, фото -> цена, уже работает
2. **Underpriced AI** — $12/мес за 30 сканов, мобильное приложение
3. **Valuify** — iOS app, подписка
4. **Value AI** — iOS app, подписка
5. **Nifty** — crosslister с оценкой цен

**Рынок НАСЫЩЕН. Минимум 5 прямых конкурентов, включая бесплатные.**

### Изюм

- Web-версия (не нужно ставить приложение)
- Бесплатно
- ...но это всё. PriceSnap делает то же самое бесплатно.

### Вирусный потенциал: НИЗКИЙ

- Результат = число. Числом не делятся в соцсетях.
- Нет "share-момента" — цена вещи это приватная информация.
- Нет эмоции в результате.

### Техническая реализация

- Требует vision model (GPT-4V, Claude Vision, Gemini Vision)
- Требует данные о ценах (парсинг маркетплейсов или API) — без этого AI будет галлюцинировать цены
- Парсинг = бэкенд обязательно, нельзя сделать single-file
- **Главная проблема:** AI без реальных данных о ценах = бесполезный. А реальные данные = парсинг = бэкенд + юридические проблемы.

## ВЕРДИКТ: Roast My Listing

**Причины:**
1. **Нет прямых конкурентов** — "roast" формат для объявлений на маркетплейсах не существует
2. **Виральность встроена** в формат — roast = делятся
3. **Single-file HTML реализуем** — текст вход, текст выход, vision не нужен
4. **AI = ядро продукта** — без AI продукт невозможен (ответ на замечание Олега)
5. **Результат = готовый текст** — пользователь копирует и вставляет, мгновенная ценность
6. **PriceTag проигрывает** по всем ключевым метрикам: насыщенный рынок, нужен бэкенд, нет вирусности
